### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、跑通管理员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 汽车信息
5. 汽车资讯
6. 汽车论坛
7. 公告信息
8. 留言板
9. 个人中心
10. 我的发布
11. 我的收藏


**普通管理员能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 汽车信息管理
6. 租车订单管理
7. 取消订单管理
8. 还车信息管理
9. 汽车资讯管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 普通管理员管理
5. 汽车类别管理
6. 汽车信息管理
7. 租车订单管理
8. 取消订单管理
9. 还车信息管理
10. 汽车资讯管理
11. 留言板管理
12. 汽车论坛
13. 系统管理（关于我们、轮播图管理、系统简介、公告信息）
